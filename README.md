# WindFlag Esp32

Fulhack av flylight. Bytt ut en esp8266 mot en esp32 med integrerad LCD-skärm i färg. Koden saknar webinterface än så länge och alla värden är hårdkodade. Lådan är omodellerad för att passa den nya skärmen och är inte helt rund längre.

Windexpilen och pinne till pilen är identisk.
